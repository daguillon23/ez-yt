# ez-yt
Utilizes yt-dlp and ffmpeg to automate the download and file conversion of YouTube videos via Python script. Requires yt-dlp and ffmpeg executables to be in same directory as Python script. Expected format of '''input.txt''' defined in Python file. Output formatted in order to more easily utilize [beets](https://beets.io/) for autotagging (though use of [MusicBrainz](https://musicbrainz.org/) still needed to manually search for appropriate release IDs).

## Tools Used
* [yt-dlp](https://github.com/yt-dlp/yt-dlp)
    * [Installation Guide (Reddit)](https://www.reddit.com/r/youtubedl/comments/qzqzaz/can_someone_please_post_a_simple_guide_on_making/)
* [ffmpeg](https://www.ffmpeg.org/)
   * [Autobuilds](https://github.com/yt-dlp/FFmpeg-Builds/wiki/Latest)
