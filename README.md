# ez-yt
Utilizes yt-dlp, ffmpeg, and YouTube API to simplify and automate the download and file conversion of YouTube videos via Python script.

## Tools Used
[yt-dlp](https://github.com/yt-dlp/yt-dlp)

[ffmpeg](https://www.ffmpeg.org/)

[Youtube API](https://developers.google.com/youtube/v3/quickstart/python)
